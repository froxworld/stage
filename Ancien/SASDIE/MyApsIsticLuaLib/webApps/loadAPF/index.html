Page de dépot de vos fichiers

1) Vous devez vous connecter :

  • Entrez votre identifiant : email de l'université
  • Entrez votre mot de passe : votre numéro d'étudiant
  • Cliquez sur "Se connecter"

2) Choisir le fichier à déposer :

  • Le choisir.
  • Lorsque ce choix est fait, le contenu du fichier apparaît en bas de la page.

3) Confirmer le dépôt :

  • Cliquez sur le bouton "Déposer".
  • La liste des fichiers déjà déposés est affichée en bas de la page.
  • Si vous déposez une nouvelle fois un fichier, l'ancienne version sera écrasée.

Cette page vous permet de déposer vos fichiers.
